# 📊 Netflix-Data-Analysis-Project
1️⃣ Project Title

Netflix Content Analysis and Visualization Dashboard

2️⃣ Project Overview

This project focuses on analyzing Netflix’s content dataset to understand trends in movies and TV shows available on the platform. The analysis includes data cleaning, exploratory data analysis (EDA), pivot table calculations, and dashboard creation to extract meaningful insights.

The goal of this project is to:

Analyze content distribution by type (Movies vs TV Shows)

Identify content trends over the years

Study country-wise content contribution

Understand genre/category distribution

Visualize key insights using an interactive dashboard

3️⃣ File Details

Original Netflix dataset file (unaltered data)

Excel sheets containing:

📁 Dashboard

Final interactive dashboard created using pivot charts and slicers

4️⃣ Data Dictionary

rating	Content rating (TV-MA, PG, etc.)

listed_in	Genre/category
